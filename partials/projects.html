Wagashi - 和菓子
English/Japanese Dictionary

Born from the need to help Japanese students solidify vocab words, users are able to search words in both Japanese and English and produce cards with the word(in Kanji), the definition, and parts of speech. Saved vocab words can be used to play a memory game called: Shinkei Suijaku (神経衰弱), or Match Up using matching pair cards. Visit the github link to check out Wagashi!

Armafit - Crossfit Scheduler

A Crossfit scheduling app that allows coaches to design and schedule WODs and users to sign up and save workouts. Users can search unfamiliar exercises and look up previous performance information. Visit the github link to checkout Armafit!
